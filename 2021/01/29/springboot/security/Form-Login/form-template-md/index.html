Spring Security Form Login - 3. 템플릿 만들기

2021-01-29
2021-03-13
Form, Spring Boot, Spring Security

Spring에서 제공하는 Thymeleaf템플릿 엔진을 이용해 로그인 페이지를 구현했다.

Form Login Page

Form 로그인을 처리하는 url로 post요청을 보낸다.

username과 password를 받는 input태그에서 name속성에 username과 password를 확실히 기입해줘야 Security가 해당 데이터를 얻어올 수 있다.

로그인 후 진입 화면

인증시에만 보이는 form태그

xmlns:sec="http://www.thymeleaf.org/extras/spring-security"을 이용하면 security를 통해 인증된 사용자에게만 보이는 화면을 구성할 수 있다.

게시판 페이지 템플릿

게시판 페이지도 게시글 작성페이지와 똑같이 Bootstrap 과 BootWatch 를 이용해 만들었다. navbar, table, pagenation 3가지 요소로 만들었다.

타임리프 문법 적용하기

  • th:each="Post:${PostList}" each구문은 반복적인 작업에서 많이 사용한다. 여기서는 데이터 리스트를 받고 각각의 데이터에 맞게 table row를 반복적으로 만들어 준다.
  • th:text="${Post.name}" 데이터를 태그 안에 들어가는 텍스트 값으로 나타낸다.
